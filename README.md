Personal Portfolio Website

A personal portfolio website showcasing my background, technical skills, projects, and experience as a software engineer.

Features

* Responsive design for desktop and mobile devices
* About Me section
* Skills overview
* Featured projects
* Experience and current focus
* Contact page
* Links to GitHub and LinkedIn

Built With

* HTML5
* CSS3
* JavaScript

Running Locally

1. Clone the repository:

git clone https://github.com/masone3/masone3.github.io.git

2. Open the project folder.
3. Open index.html in your browser, or use a local development server such as the VS Code Live Server extension.
